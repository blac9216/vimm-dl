# Progress Comment Template

Post as a comment on the issue you are working so a human or another agent can pick
up where you left off.

```markdown
## Progress Update

**Status**: investigating | in-progress | testing | blocked

### What was done
- Step-by-step list of actions taken

### Findings
- What was learned from each step

### Current state
- Where things stand right now

### Next steps
- What remains to be done

### Blockers (if any)
- What is preventing progress and what help is needed
```
